# Система Розрахунку Кошторису v4.5

Професійний інструмент для автоматизації розрахунків обладнання та послуг.

## Основні можливості:
*   **База Даних**: Окреме керування асортиментом товарів та послуг.
*   **Розумний пошук**: Миттєва фільтрація бази за назвою або категорією.
*   **Експорт**: Професійні PDF та Excel звіти з групуванням за категоріями та алфавітним сортуванням.
*   **Історія**: Збереження проектів з точним часом та можливістю повторного завантаження.
*   **Персоналізація**: Налаштування кольорів інтерфейсу (Хедер/Футер) та підтримка темної теми.
*   **Податки**: Гнучке налаштування ставки ПДВ.

## Технології:
*   WPF (.NET 8.0)
*   Material Design in XAML
*   MVVM Toolkit
*   QuestPDF & ClosedXML

## Як користуватися:
1. Запустіть `EstimateApp.exe`.
2. Наповніть "Базу Даних".
3. У розділі "Розрахунок" виберіть потрібні позиції та вкажіть кількість.
4. Натисніть "Зберегти в історію" або експортуйте у PDF/Excel.
